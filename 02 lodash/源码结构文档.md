# Lodash 源码结构文档

## 项目概述

**项目名称**: lodash
**版本**: 4.18.1
**架构类型**: 单体单文件架构 (Monolithic Single-File)
**许可证**: MIT
**源码规模**: 约 17,572 行代码（翻译后）

Lodash 是一个功能强大的 JavaScript 实用工具库，提供模块化、高性能的数组、对象、字符串、数字等操作函数。本项目采用单体单文件架构，所有功能都包含在 `lodash.js` 文件中。

## 核心类实现

### 缓存类 (行号 ~1700-2300)

| 类名 | 用途 | 内部结构 |
|------|------|---------|
| Hash | 哈希表实现 | key/value 映射 |
| ListCache | 列表缓存 | 数组+哈希混合 |
| MapCache | Map 缓存 | 支持 Map 所有操作 |
| SetCache | Set 缓存 | 支持 Set 所有操作 |
| Stack | 栈结构 | 用于深度遍历 |

### 延迟包装器

```
LazyWrapper (行号 ~2100-2250)
├── __wrapped__   原始值
├── __actions__   待执行动作
├── __dir__      遍历方向 (1/-1)
├── __filtered__ 是否已过滤
├── __iteratees__ 迭代器列表
├── __takeCount__ 需要获取元素数
└── __views__    视图信息
```

## 构建系统

### 构建脚本 (lib/)

| 路径 | 文件 | 作用 |
|------|------|------|
| common/ | file.js | 文件操作 |
| common/ | mapping.js | 方法映射 |
| common/ | minify.js | 代码压缩 |
| main/ | build-dist.js | 构建分发版本 |
| fp/ | build-dist.js | 构建FP版本 |

### 构建流程

```
npm run build
├── build:main
│   └── node lib/main/build-dist.js
│       ├── lodash.js → dist/lodash.js
│       └── lodash.min.js (压缩)
└── build:fp
    └── node lib/fp/build-dist.js
        └── lodash.fp.js (柯里化版本)
```

## 开发指南

### 常用命令

```bash
# 构建
npm run build              # 构建所有版本
npm run build:main         # 仅构建主版本
npm run build:fp           # 仅构建FP版本

# 测试
npm test                   # 运行所有测试
npm run test:main          # 主功能测试
npm run test:fp            # FP功能测试
```
